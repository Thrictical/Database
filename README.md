# Contact Directory

A simple and efficient way to manage your contact list. Built with React (TypeScript), Express, and SQLite.

## Features
- ✨ Create, Read, Update, and Delete contacts.
- 📱 Mobile-friendly design.
- 🔍 Search functionality.
- 🎨 Modern glassmorphism UI.

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Thrictical/Database.git
   cd Database
   ```

2. **Setup the Server:**
   ```bash
   cd server
   npm install
   npm run dev
   ```

3. **Setup the Client:**
   ```bash
   cd ../client
   npm install
   npm run dev
   ```

### Usage
- The client will run at `http://localhost:5173`.
- The server will run at `http://localhost:5000`.

## Tech Stack
- **Frontend:** React, TypeScript, Vite, CSS
- **Backend:** Node.js, Express, ts-node-dev
- **Database:** SQLite (better-sqlite3)
